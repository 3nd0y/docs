Documentation

Learn how to grow the Network and connect all Things!

The Things Network involves Devices like The Things Uno and The Things Node to Connect all Things, gateways like The Things Gateway to Grow the Network these devices connect to and finally The Things Network Dashboard. This where you Manage how Devices Integrate with Applications on the Internet via the network.


You can contribute to this official documentation via GitHub.

For other community contributed content, please use the Wiki.

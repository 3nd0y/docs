Account Server

The Account Server manages the ACL for The Things Network. It closed-source. If you’d like to run the open-source backend you will need to provide an account server with the exact same API.

This guide will inform you about the access and use the Account Server API.

The endpoint for The Things Network Account Servers is currently https://account.thethingsnetwork.org.
